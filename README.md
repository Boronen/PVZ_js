# Warlords: Last Siege

> A browser-based 2D singleplayer lane-battler roguelike strategy game.

---

## 🎮 About

**Warlords: Last Siege** blends the lane-combat loop of *Age of War*, the run-based upgrade variety of *Vampire Survivors*, and the chaotic synergy-building of *The Binding of Isaac*.

Select a hero commander, spawn units across three tiers, collect roguelike upgrades that stack and interact, and destroy the AI-controlled enemy base before your own falls.

## 🚀 Run Locally

```bash
git clone https://github.com/<your-org>/warlords-last-siege.git
cd warlords-last-siege
npm install
npm run dev
# Open http://localhost:8080
```

> The game requires a web server — do not open `index.html` directly from the file system.

---

## 🧪 Run Tests

```bash
npm run dev          # Start server (terminal 1)
npm run test:open    # Interactive Cypress (terminal 2)
npm run test         # Headless CI mode
```
